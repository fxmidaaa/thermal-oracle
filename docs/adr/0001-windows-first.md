# ADR-0001: Запуск Windows-first

Дата: 2026-06-11 · Статус: принято (решение владельца продукта)

## Контекст
Двухплатформенный агент (LibreHardwareMonitor на Windows, powermetrics на
macOS) удваивает поверхность тестирования. На Apple Silicon температура
кристалла через powermetrics доступна ограниченно, что урезает ценность
продукта на этой платформе до решения проблемы.

## Решение
v1 — строго Windows + LibreHardwareMonitor JSON API. Механизм `capabilities`
(декларация доступных метрик при регистрации устройства) уже заложен в схему
ingest и БД, чтобы macOS-агент в v2 подключился без миграций.

## Последствия
- Агент v1 содержит единственный коллектор `windows_lhm`; интерфейс
  `collectors/base.py` сохраняется для будущего `macos_powermetrics`.
- Аналитика обязана уважать `capabilities` с первого дня (минимум для Rth:
  температура + мощность).
